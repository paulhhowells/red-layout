A narrow (e.g. mobile) layout exists until a viewport width of about 28 or 30em, and then a wider layout takes over. There is a gap between 28 and 30em (which should be removed in production) in order to watch the transfer between breakpoints while developing.

At narrow widths a button is provided for showing and hiding the side drawer which will contain navigation.
